# Legacy Docs Quarantine

Folder ini adalah tempat karantina dokumen lama dari proyek RKK. 

## Aturan Penting
- Dokumen di sini belum tentu dipakai sebagai dokumentasi aktif.
- **JANGAN** menghapus isi folder ini sebelum informasi pentingnya dimigrasikan ke dokumentasi aktif.
- Setiap file yang dipindah ke sini **harus** dicatat di migration tracker (`docs/project/migration/LEGACY_DOCS_INVENTORY.md`).
- Penghapusan dokumen di sini hanya boleh dilakukan pada batch *cleanup* khusus setelah mendapatkan persetujuan eksplisit dari User.

## Struktur
- `original-docs/` - Berisi file dan folder dokumentasi lama dengan mempertahankan struktur aslinya.
