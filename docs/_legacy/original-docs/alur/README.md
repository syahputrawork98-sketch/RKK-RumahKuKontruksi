# Alur Bisnis RKK - RumahKu Konstruksi

Folder ini berisi dokumentasi alur bisnis aktif yang digunakan dalam pengembangan aplikasi RKK (RumahKu Konstruksi) pada fase **Local Development Feature Completion**.

## 📑 Daftar Dokumen

### Alur Induk
- [**Alur Bisnis RKK**](./alur-bisnis-rkk.md): Penjelasan mendalam alur end-to-end dari fase Publik hingga Project Closeout.

### Alur Per Role (Detail)
- [**Alur Publik**](./alur-publik.md): Fokus pada landing page dan konversi ke konsumen.
- [**Alur Konsumen**](./alur-konsumen.md): Fokus pada pengajuan desain dan monitoring proyek.
- [**Alur Superadmin**](./alur-superadmin.md): Fokus pada manajemen persona dan tata kelola sistem.
- [**Alur Admin**](./alur-admin.md): Fokus pada konversi desain, RAB, dan moderasi operasional.
- [**Alur Arsitek**](./alur-arsitek.md): Fokus pada proses tender, bidding, dan desain aktif.
- [**Alur Pengawas**](./alur-pengawas.md): Fokus pada verifikasi progres fisik (SOT) dan review lapangan.
- [**Alur Mandor**](./alur-mandor.md): Fokus pada pelaksanaan fisik dan pelaporan harian/mingguan.

## ⚠️ Catatan Penting
- Dokumen ini merujuk pada kondisi database, API, dan UI **aktif** saat ini di repositori.
- Dokumen ini digunakan untuk menyelaraskan pemahaman alur bisnis lintas role.
- **Bukan SOP Agent**: Untuk instruksi pengerjaan teknis oleh AI Agent, silakan merujuk ke [`docs/project/workflow-sop.md`](../project/workflow-sop.md).
- **Legacy Reference**: Dokumen di folder `docs/archive/` atau referensi bisnis lama hanya digunakan sebagai pembanding dan **bukan** merupakan *Source of Truth* aktif jika bertentangan dengan dokumen di folder ini.
