# Existing Frontend Structure

Kondisi existing struktur frontend pada proyek RKK.

## Teknologi Utama
- Ditempatkan dalam direktori `client/`.
- Dibangun menggunakan **React** dan build tool **Vite**.

## Fungsionalitas Dikenali
- Proyek mendukung routing untuk berbagai role, antara lain:
  - Public
  - Konsumen
  - Superadmin
  - Admin
  - Pengawas
  - Mandor

*Catatan Baseline: Dokumentasi ini tidak bertujuan menilai terlalu dalam mengenai kualitas atau best practices dari kode yang ada, melainkan sekadar mencatat sebagai referensi awal kondisi existing.*
