# Catatan Teknis Backend RKK

Folder ini dikhususkan untuk dokumentasi terkait pengembangan backend API proyek RKK.

## Informasi Dasar
- Backend API RKK berada di direktori `server/`.
- Dibangun menggunakan **Node.js** dan **Express**.
- Mengatur endpoint route untuk `project`, `admin`, `pengawas`, dan `mandor`.

## Aturan Pengembangan
- Dokumentasi struktur endpoint secara detail dan spesifikasi request/response API akan ditambahkan pada batch-batch berikutnya.
- Hindari menyentuh file fungsionalitas lain ketika sedang mengerjakan perbaikan spesifik di satu endpoint.
