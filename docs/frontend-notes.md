# Frontend Notes

## Stack Teknologi
- **Build Tool**: Vite
- **Framework/UI Library**: React
- **Routing**: `react-router-dom`
- **Styling**: Tailwind CSS & DaisyUI
- **Sumber Frontend**: `RumahKuKontruksi-Dev/client`

## Struktur Folder `client/`
- `src/`: Komponen, modules, layouts, dan pages.
- `public/`: Aset statis.
- `index.html`: Entry point.
- `vite.config.js`: Konfigurasi Vite.

## Role Utama
- Guest (Halaman Umum)
- Konsumen
- Superadmin
- Admin
- Pengawas
- Mandor

## Cara Menjalankan
1. Masuk ke folder client: `cd client`
2. Install dependensi: `npm install`
3. Jalankan mode dev: `npm run dev`
