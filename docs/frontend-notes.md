# Frontend Notes

## Stack Teknologi
- **Framework**: Next.js 16.1.6
- **UI Library**: React 19.2.3
- **Styling**: Tailwind CSS 4
- **Language**: TypeScript

## Struktur Folder `client/`
- `app/`: Next.js App Router.
- `src/`: Komponen, hooks, dan utilitas.
- `public/`: Aset statis.
- `openapi/`: Definisi API (Swagger/OpenAPI).

## Cara Menjalankan
1. Masuk ke folder client: `cd client`
2. Install dependensi: `npm install`
3. Jalankan mode dev: `npm run dev`
