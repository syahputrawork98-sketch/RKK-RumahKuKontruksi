# Legacy Docs Cleanup Plan

Dokumen ini menjelaskan rencana cleanup bertahap untuk dokumen lama yang sudah dikarantina. 

*Catatan Penting: Tidak ada file legacy yang boleh dihapus sebelum statusnya menjadi `Migrated`, `Duplicate`, `Deprecated`, atau `Safe to Delete Later`, dan User menyetujui eksekusi tersebut.*

## Tahapan Cleanup
- **F01C** = Review isi legacy docs.
- **F01D** = Migrasi informasi penting ke docs aktif.
- **F01E.1** = Hapus file legacy yang sudah migrated/safe.
- **F01E.2** = Hapus batch legacy berikutnya.
- **F01E-CP** = Checkpoint dokumentasi setelah cleanup selesai.
